Install: 

$ gem install slim

In your Gemfile (Rails) 

gem 'slim', require: 'slim/rails'

Now on to the syntax...

Slim escapses html when calling a method: 

= some_method
Rails provides an html_safe? method that Slim takes advantage of, but when rendering partials, you'll need to use the double equal to bypass the escape html method: 
== render :partial, 'user'

Haml like shortcuts: 

#myid.myclass.myclass2 My example
renders: 
<div id="myid" class="myclass myclass2">My example</div>

Code comments (use a forward slash): 

/ This is my comment, it will not be included on output
p This is my content

Span multiple lines (use a back slash): 

= javascript_include_tag 'jquery', \
  'application'
p This is my content
